# Mai Hỷ Tea House Menu Website 🍵

A beautiful, responsive website for Mai Hỷ Tea House (梅喜) - an elegant Vietnamese tea establishment located at 142 Hàng Bông, Hoàn Kiếm, Hà Nội.

## Features

✨ **Elegant Design**
- Sophisticated typography with Cormorant Garamond serif and DM Sans fonts
- Refined color palette: warm neutrals and earth tones
- Professional layout showcasing tea varieties and pricing

📱 **Responsive Layout**
- Mobile-friendly design
- Grid-based layout that adapts to different screen sizes
- Optimized navigation and content display

☕ **Menu Sections**
- **Cổ thụ tinh tuyển** (Premium Ancient Tree Teas)
  - Móng Rồng
  - Lục Bưởi
  - Hồng Suối Giàng
  - Hồng Trà Quýt

- **Tinh hoa trà Việt** (Premium Vietnamese Teas)
  - Olong Mộc Tê
  - Olong Cúc
  - Earl Grey
  - Lục Nhài

- **Matcha Việt Nam** (Vietnamese Matcha)
  - Matcha Latte
  - Matcha Oatside
  - Matcha Hồng Quýt

## Service Types

1. **Theo ấm** - 5g per pot, traditional self-brewing
2. **Theo tuần** - Large pot, full experience
3. **Ủ lạnh** - Cold brew overnight method

## How to Use

1. Clone the repository or download the files
2. Open `index.html` in your web browser
3. The website is fully responsive and works on desktop, tablet, and mobile devices

## Customization

To customize the website:
- Edit tea names and descriptions in the HTML
- Update prices in the price sections
- Modify contact information as needed
- Replace images by updating image URLs
- Adjust colors by modifying the hex values in the inline styles

## Browser Support

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
